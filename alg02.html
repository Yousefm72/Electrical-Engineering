האם אפשר ש-CA · BD = 0 כששניהם באותו כיוון?

התשובה הקצרה

כן. המכפלה הסקלרית של שני וקטורים באותו כיוון יכולה להיות שווה לאפס אך ורק כאשר לפחות אחד הוקטורים הוא וקטור האפס (כלומר גודלו 0).

עיקרון: אם הוקטורים מקבילים (θ = 0° או 180°) ואינם אפס, אז:

u · v = |u| |v| cos(θ) = ± |u| |v| ≠ 0

לכן כדי לקבל 0 נדרש |u| = 0 או |v| = 0.

רקע: מכפלה סקלרית

המכפלה הסקלרית בין שני וקטורים u ו-v מוגדרת כך:

u · v = ux vx + uy vy = |u| |v| cos(θ)

כאשר θ היא הזווית בין הוקטורים. אם הוקטורים באותו כיוון → θ = 0° (או 180° אם בכיוונים הפוכים, ועדיין מקבילים).

יישום לשאלה עם הוקטורים CA ו-BD

נסמן CA→ = A − C ו-BD→ = D − B. אם הוקטורים מקבילים (אותו כיוון או כיוון הפוך), אז:

CA→ · BD→ = |CA→| |BD→| cos(θ) = ± |CA→| |BD→|

מכאן ש-CA→ · BD→ = 0 יקרה רק כאשר |CA→| = 0 או |BD→| = 0. כלומר, כאשר לפחות אחד הוקטורים הוא וקטור האפס:

  • CA→ = 0 כאשר C = A (אותה נקודה).
  • BD→ = 0 כאשר B = D (אותה נקודה).

דוגמאות

דוגמה 1: וקטור אפס מול וקטור לא-אפס

נניח C = A = (1, 2) ⇒ CA→ = (0, 0). כמו כן, B = (0, 0), D = (3, 4) ⇒ BD→ = (3, 4).

CA→ · BD→ = (0, 0) · (3, 4) = 0

הוקטורים “באותו כיוון” באופן טריוויאלי (ל-CA→ אין כיוון מוגדר כי הוא אפס), והתוצאה אפס.

דוגמה 2: וקטורים מקבילים ולא-אפסיים

נניח CA→ = (2, 2) ו-BD→ = (5, 5). הוקטורים מקבילים ובאותו כיוון.

CA→ · BD→ = 2·5 + 2·5 = 20 ≠ 0

כאן המכפלה אינה אפס, כי שני הוקטורים בעלי גודל חיובי.

דוגמה 3: מקבילים בכיוונים הפוכים

CA→ = (2, 2), BD→ = (−5, −5). הוקטורים מקבילים אך הפוכים בכיוון (θ = 180°).

CA→ · BD→ = 2·(−5) + 2·(−5) = −20 ≠ 0

עדיין לא אפס. שוב, כדי לקבל אפס צריך שאחד מהם יהיה וקטור אפס.

איור אינטואיטיבי

כאשר אחד הוקטורים הוא אפס, המכפלה הסקלרית שווה לאפס – גם אם הוקטורים "באותו כיוון".

לסיכום

  • אם CA→ ו-BD→ מקבילים ואינם אפס → המכפלה הסקלרית אינה אפס.
  • המכפלה יכולה להיות אפס רק אם CA→ = 0 או BD→ = 0.
  • וקטור אפס מתקבל כאשר נקודות ההתחלה והסיום שלו חופפות (למשל C = A או B = D).
