אם BA · BD < 0 ⇒ הזווית בין BA ו-BD קהה

ניסוח השאלה

המשפט הנתון: אם המכפלה הסקלרית BA→ · BD→ שלילית (כלומר BA→ · BD→ < 0), אז הזווית בין הישרים BA ו-BD היא זווית קהה.

רקע קצר: מכפלה סקלרית וזווית

הקשר הבסיסי בין מכפלה סקלרית לזווית θ בין שני וקטורים u, v:

u · v = |u| |v| cos(θ)

מכאן נובע:

  • u · v > 0 ⇒ cos(θ) > 0 ⇒ הזווית חדה (0°–90°).
  • u · v = 0 ⇒ cos(θ) = 0 ⇒ הזווית ישרה (90°).
  • u · v < 0 ⇒ cos(θ) < 0 ⇒ הזווית קהה (90°–180°).

הסבר ופתרון לטענה

נסמן את הוקטורים: BA→ = A − B, BD→ = D − B. אם BA→ · BD→ < 0, אז לפי הנוסחה:

BA→ · BD→ = |BA→| |BD→| cos(θ)

מכיוון ש-|BA→| ≥ 0 ו-|BD→| ≥ 0, הסימן של המכפלה נקבע על-ידי cos(θ) בלבד. לכן cos(θ) < 0 ⇒ 90° < θ < 180°. כלומר הזווית בין הישרים קהה.

מסקנה: BA→ · BD→ < 0 ⇔ הזווית בין BA ו-BD קהה.

דוגמאות מספריות

דוגמה 1: מכפלה שלילית ⇒ זווית קהה

נניח BA→ = (1, 0), BD→ = (−1, 1).

BA→ · BD→ = 1·(−1) + 0·1 = −1 < 0

הזווית בין הוקטורים גדולה מ-90° (בפועל 135°) – קהה.

דוגמה 2: מכפלה חיובית ⇒ זווית חדה

BA→ = (1, 0), BD→ = (1, 1).

BA→ · BD→ = 1·1 + 0·1 = 1 > 0

הזווית קטנה מ-90° (45°) – חדה.

דוגמה 3: מכפלה אפס ⇒ זווית ישרה

BA→ = (0, 2), BD→ = (3, 0).

BA→ · BD→ = 0·3 + 2·0 = 0

הזווית שווה ל-90° – ישרה.

איור אינטואיטיבי

כאשר BA→ · BD→ < 0, הקוסינוס של הזווית שלילי ולכן הזווית קהה.

לסיכום

  • BA→ · BD→ < 0 ⇔ 90° < θ < 180° בין BA ו-BD.
  • הסימן של המכפלה הסקלרית מוכתב על-ידי cos(θ).
  • כלל אצבע: שלילי → קהה, אפס → ישרה, חיובי → חדה.
